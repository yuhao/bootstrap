About

Computer architecture research has been largely driven by access and awareness of tools and other infrastructure. In this tutorial, we present a series of tools and other infrastructure that enables computer architecture researches in the mobile computing space. The tutorial will spans three key components of mobile computing: software, hardware, and end-users. The tools that we will cover strongly emphasize real system measurement, including performance, power/energy, user QoS experience.
